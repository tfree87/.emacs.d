1. AUCTeX   LaTeX

Use AUCTeX for writing documents using \LaTeX{}. 

(use-package auctex
  :straight t
  :defer t
  :custom
  (TeX-auto-save t)
  (TeX-parse-self t)
  (TeX-master nil)
  (bibtex-dialect 'biblatex))

1.1. Use BibLaTeX

BibLaTeX has more powerful features than standard BibTeX. 

(bibtex-dialect 'biblatex)
