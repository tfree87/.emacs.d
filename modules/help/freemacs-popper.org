* Popper

[[https://github.com/karthink/popper][Popper]] allows for certain buffers to display in popup windows by default.

#+begin_src emacs-lisp :tangle yes :noweb yes
  ;; -*- lexical-binding: t -*-

  (use-package popper
    :straight t
    :bind (("C-`"   . popper-toggle-latest)
           ("M-`"   . popper-cycle)
           ("C-M-`" . popper-toggle-type))
    :custeom
    <<popper_custom>>
    :config
    (popper-mode t)
    (popper-echo-mode t))


  (require 'freemacs-popper)

  ;;; freemacs-popper.el ends here.
#+end_src

** Popper Buffers

For each buffer that will become a popper pop up buffer, it must be defined below.
#+begin_src emacs-lisp :noweb-ref popper_custom
  (popper-reference-buffers
          '("\\*Warnings\\*"
            "\\*Messages\\*"
            "\\*Embark Actions\\*"
            "Output\\*$"
            "\\*Async Shell Command\\*"
            "\\*Whicher Report\\*"
            help-mode
            compilation-mode))
#+end_src
